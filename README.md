# Nvim-Config
My personel configuration of Neovim utilizing:
- Treesitter
- Packer
- Harpoon
- LSP-0
- Custom KeyMaps

## Dependencies:
 - Neovim Version 0.9+
 -- [Building Neovim](https://github.com/neovim/neovim/wiki/Building-Neovim)
 

## Installation
```
cd ~/.config && git clone https://github.com/JnathanCarter/Nvim-Config nvim
```
